# UVL Sentinel (MODEVAR 24)

## Python environment

```
python -m venv env
```

### Activate (Windows)

```
env\Scripts\activate
```

### Activate (Linux)

```
source env/bin/activate
```

## Install

```
make install
```

## Working with UVL Sentinel

### Generate analysis

```
python main.py
```

### Generate summary

```
python summary.py
```
